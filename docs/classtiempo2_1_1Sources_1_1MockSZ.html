Detailed Description

Class to represent the Sunyaev-Zeldovich effect on-sky.

In essence, this class wraps main functionality of MockSZ. It generates an isothermal-beta model and simulates the tSZ and kSZ effects, as function of angle on-sky. As output, it generates two datacubes (one for tSZ and kSZ) with the following layout: (dAzimuth, dElevation, frequency). Here, dAzimuth is the azimuthal angle from cluster center, dElevation elevation angle from cluster center and frequency the frequency of radiation.

Parameters
TeElectron temperature of cluster plasma, in KeV.
ne0Central electron number density in 1 / cm**3.
rcCluster core radius in kpc.
betaStructure constant of cluster in isothermal-beta model.
DaCluster angular diameter distance in Mpc.
v_pecPeculiar cluster velocity in km /s.

Generate SZ maps on-sky over a rage of frequencies.

Note that the lower and upper azimuth and elevation angles are with respect to cluster center, i.e., centered around 0.

Parameters
freq_HzRange of frequencies over which to generate maps, in Hertz.
lims_AzLower and upper limits on azimuth range in degrees.
lims_ElLower and upper limits on elevation range in degrees.
nAzNumber of cells along azimuth axis.
nElNumber of cells along elevation axis.
ret_unitReturn maps in this unit. Choose between standard W m**-2 Hz**-1 sr**-1 ('SI'), MJy sr**-1 ('MJy') or mK ('mK').
Returns
tSZ Datacube containing tSZ intensity sky maps across the frequency range.
kSZ Datacube containing kSZ intensity sky maps across the frequency range.
