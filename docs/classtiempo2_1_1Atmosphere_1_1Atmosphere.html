Detailed Description

Class that represents atmosphere.

Uses ARIS data to generate PWV matrix.

This class handles generation of PWV maps from ARIS data. It sets the path to the FITS files, sets the relevant parameters, such as map size, cell size, observation duration, and wind speed. Also, the chopper angular throw, scanned area on-sky and telescope elevation are set here.
